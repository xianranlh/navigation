# 个人导航页面

一个简洁、美观的个人导航页面，帮助您快速访问常用网站。

## 功能特性

- 🎨 **主题切换**：支持明暗主题切换，保护您的眼睛
- 🔍 **智能搜索**：集成百度、Google、Bing三大搜索引擎
- 📱 **响应式设计**：完美适配桌面端和移动端
- ⚡ **快捷操作**：支持键盘快捷键（`/` 聚焦搜索，`Esc` 清空）
- 💾 **本地存储**：自动保存主题和搜索引擎偏好
- 🎯 **分类导航**：常用网站、社交媒体、开发工具、视频娱乐、在线工具等分类

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或者部署到任何静态网站托管服务（如 GitHub Pages、Netlify、Vercel 等）

## 快捷键

- `/` - 聚焦搜索框
- `Esc` - 清空搜索框并失焦
- `Enter` - 执行搜索

## 自定义

您可以通过编辑 `index.html` 文件来添加或修改导航链接：

```html
<a href="https://example.com" class="link-card" target="_blank">
    <div class="link-icon">图标</div>
    <span class="link-name">网站名称</span>
</a>
```
