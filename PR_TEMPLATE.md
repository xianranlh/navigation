## ✅ 代码变更检查清单
请提交PR前，确认已完成以下事项：

### 🛠️ 通用检查
- [ ] 代码已合并最新的 `main` 分支，无冲突
- [ ] 本次变更已在本地运行 `npm run dev` 测试，无报错
- [ ] 已检查控制台（Console）无警告或错误输出
- [ ] 已移除或注释所有调试代码（如 `console.log`、调试断点）

### 🎨 前端与样式检查 (Next.js / Tailwind)
- [ ] 页面响应式布局在移动端和桌面端显示正常
- [ ] 新增或修改的组件在**深色/浅色模式**下均经过测试
- [ ] 没有直接覆盖或硬编码与Tailwind冲突的CSS样式
- [ ] 使用的图标（Lucide React）尺寸和颜色符合上下文

### 🗃️ 数据层检查 (Prisma / SQLite)
- [ ] 如果修改了 `prisma/schema.prisma`，已执行 `npx prisma generate`
- [ ] 数据库迁移文件（如有）已创建且可以安全回滚
- [ ] 对数据库的查询已考虑性能（如添加了必要的索引）

### 🔒 安全检查
- [ ] 没有将敏感信息（如密码、密钥）硬编码在代码中
- [ ] 所有用户输入（如表单）都已做适当的验证或转义

### 📚 文档与用户体验
- [ ] 更新了相关的README或内联代码注释（如有必要）
- [ ] 新增功能的用户界面有清晰的引导或提示
